# OVO — AI Outfit App

## Overview
AI-powered outfit recommendation and virtual try-on mobile app.

## Tech Stack
- Backend: ABP.io (.NET)
- Mobile: React Native (Expo)
- AI: Banana Pro
- Storage: S3 / Azure Blob

## Core Modules
- Auth (OTP, SMS)
- Wardrobe
- Try-On
- Community
- Outfit Engine

## Architecture
- Modular Monolith (ABP) → later microservice
- Clean Architecture
- DDD
